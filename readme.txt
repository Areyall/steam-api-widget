== Description ==

This plugin will add a widget that displays your Steam profile and recent played games.

features:

 - Steam profile
 - Recent played games
 - Bootstrap CSS Framework
 - Customizable

Visit http://8bit-life.com/steam-wordpress-plugin/

== Installation ==


Upload the Steam-Api-Widget plugin to your blog
Activate it then enter your API key and the SteamID64
If you don't know your SteamID64, this tool helps: http://steamidconverter.com/
You'll need an API key (http://steamcommunity.com/dev/apikey) to use it. 
Keys are free.

You're done!
